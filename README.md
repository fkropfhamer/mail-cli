# Command line mailing program

- requires: node, yarn

* -h HELP!!!
* -m [path to msg.txt]
* -p // no arg | send mailing no test mode
* -s [subject]
* -r [path to receivers.csv]


### Getting started 

- `yarn`
- replace content in `_config.json` and rename to `config.json`
- use like `node src/main -h`
